# TaskFlow – Full Stack Task Management System

## Overview
TaskFlow is a full-stack task management application built using the MERN stack. It allows users to create, update, and manage tasks efficiently.

## Features
- Create, update, and delete tasks
- User-friendly dashboard
- Full-stack MERN architecture
- Responsive UI

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## Installation
1. Clone the repository
2. Install dependencies
3. Run client and server
